A simple and interactive **Daily Expense Tracker** web app built using **HTML, CSS, and JavaScript**.  
It helps users easily record and monitor their daily spending habits.

---

##  Features

- Add new expenses with **amount, category, and date**
- Automatically calculates **total spending**
- Filter expenses by category
- Simple, responsive, and user-friendly interface
- Data stored temporarily (in memory)
